# Reading Coach

영어 독해에서 자주 흔들리는 포인트를 집중 훈련하는 로컬 웹앱입니다.

- 주체와 객체를 바꿔 묻는 문제
- 한 문장 다음의 흐름을 추론하는 문제
- 시제와 적용 시점을 섞어 놓은 이중지문 함정

## 구성

- `index.html`: 화면 레이아웃
- `styles.css`: 스타일
- `app.js`: 문제 생성 요청, 렌더링, 채점
- `server.py`: 정적 파일 제공 + OpenAI API 프록시

## 실행 방법

1. Python 3를 설치합니다.
2. 작업 폴더에서 아래 명령을 실행합니다.

```powershell
python server.py
```

3. 브라우저에서 `http://127.0.0.1:8000` 을 엽니다.
4. OpenAI API 키를 넣고 문제를 생성합니다.

## 메모

- API 키는 브라우저 저장소에 저장하지 않고 생성 요청 때만 로컬 서버로 보냅니다.
- 이 프로젝트는 표준 라이브러리만 사용해서 시작 비용을 낮췄습니다.
- 나중에는 오답 기록, 약점별 재출제, CSV 저장, 토익 파트별 템플릿 확장도 쉽게 붙일 수 있습니다.
